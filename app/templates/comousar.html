Manual :


Login :

Para Realizar o login, deverar usar o nome de usuário e a senha fornecidos pelo administrador e/ou coondenador do curso.
Caso tenha esquecido a senha a mesma poderar se recuperada no campo esquecir a senha na tela de login.

Telas :

1 - Início:

  • Tela principal onde o usuário poderá ter acesso as demais tela e logo do sistema

2 - Cadastros:

  • Tela disponivel apenas para usuarios administradores do sistema
  • Ao clicar no botão cadastro será direcionado para a tela de listagem de usuários onde irá aparecer todos o usuarios cadastrados
  • Dentro da tela de listagem terá um botão de "Novo" para cadastrar novos usuários e o botão de "Voltar" que será direcionado para a tela principal
  • Ao clicar em quaisquer parte do dados que aparece na tela de listagem, será direcionado para a tela de edição, onde será possível editar e excluir o cadastro do usuario selecionado
  • Todos os dados editados ou excluidos não poderão mais serem recuperados
  • Ao final de cada operação de inserção e alteração nos dados o sistema irá retornar uma mensagem de confimação

3 - Situações:

  • Tela para inserir a situaçõe de cada disciplina de um determinado aluno
  • Deverá selecionar a disciplina e depois a situação da mesma, sendo:
    • Aprovado - Para Alunos que tenha sido aprovado na disciplina escolhida
    • Reprovado - Para Alunos que tenha sido reprovado na disciplina escolhida
    • Matricula - Para Alunos que ainda estão fazendo a disciplina escolhida
    • Desistente - Para Alunos que desistiu/trancou a disciplina escolhida
  • Para cada disciplina selecionada deverá clicar no botão "Inserir"
  • Para finalizar as inserções e gerar as probabilidades deverá clicar no botão "Finalizar"
  • Após clicar no botão Finalizar irá carregar uma tela com os resultados na analise, as probabilidades de Aprovação é Reprovação daquele aluno, para de acordo com aquela disciplina.
  • Na tela de resultados das analise, tem a opção para salvar os dados da análise
  • Para salvar so dados basta inserir o cpf do aluno e clicar em salvar

    • * A análise deverá ser feita para cada discilina por vez.
    * Não sendo permitido fazer a análise de mesma disiciplina para o mesmo aluno.

4 - Relatórios:

    No menu relatorio é possivel saber as seguintes situações:
  • Alunos com análise salva: Onde listará todos os alunos que tiveram suas análises salvas na tela de Situações
  • Alunos com risco de reprovação igual ou maior de 60%: Listará todos os alunos que possui um risco de reprovação no curso.
  • Percentual por disciplinas: Inserindo o CPF do aluno que podera lista a sistuação de cada disciplina separado
  • Relação dos usuarios cadastrados no sistema
  • Relação das diciplinas do curso de TADS
  • Relação das disciplinas com a maior nivel de aprovação
  • Relação das disciplinas com a maior nivel de reprovação
  • Relação da média de matricula por disciplina
  • Relação das disciplinas com maiores indice de desistência

  • Essa tela explica o conteúdo de cada intem do menu
